The Insertion Sort algorithm has been fully completed with metrics with no issues or bugs.

The Merge Sort algorithm has also been fully completed with metrics with no issues or bugs.

-------------------------------------------------------------------------------------------

Insertion Sort Analysis

Best Case (Already Sorted): O(n), n comparisons, 0 swaps, 0 copys
Average Case: O(n^2), (n(n-1)/2) comparisons, (n(n-1)/2) swaps, (n-1) copys
Worst Case (Reverse Sorted): O(n^2), (n(n-1)/2) comparisons, (n(n-1)/2) swaps, (n-1) copys

Merge Sort Analysis

Best Case (Already Sorted): O(nlogn), nlogn comparisons, 0 swaps, 2(nlogn) copys
Average Case: O(nlogn), nlogn comparisons, 0 swaps, 2(nlogn) copys
Worst Case (Reverse Sorted): O(nlogn),  nlogn comparisons, 0 swaps, 2(nlogn) copys
